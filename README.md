# LYSA Project

**LYSA** est une plateforme WEB pour l'Internet des Objets (**IOT**). Le but de ce système est de fournir une interface agréable et intuitive pour la gestion d'objets. Ces derniers sont en fait des capteurs, des actionneurs ou tout type d'objets connectés qui nécessite un suivi.

> C'est une solution très simple et sécurisé.  
> L'administrateur garde le contrôle total de son accés Internet.  
> Cette solution est entièrement Open Source.
